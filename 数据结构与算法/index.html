数据结构与算法

时间复杂度

  • O(1):常数复杂度

      int n= 1000;
  System.out.print("n:"+n)

  • O(log n):对数复杂度

      for(int i=;i<n;i=i*2){
      System.out.print("n:"+n)
  }

  • O(n):线性时间复杂度

      for(int i=0;i<n:i++){
      System.out.print("i:"+i)
  }

  • O(n^2):平方

      for(int i=0;i<n:i++){
      for(int j=0;j<n;j++){
          System.out.print("i:"+i + "j:"+j)
      }
  }

  • O(n^3):立方

      for(int i=0;i<Math.pow(n,3);i++){
      System.out.print("i:"+i)
  }

  • O(2^n):指数

      for(int i=0;i<Math.pow(2,n);i++){
      System.out.print("i:"+i)
  }

从上到下，时间越来越慢！！！

主定理

算法 时间复杂度
二分搜寻算法 O(log n)
二叉树遍历 O(n)
最佳排序矩阵搜索 O(n)
合并排序 O(n*logn)

算法导论主定理

Array

  • 连续内存空间；
  • 查询操作指定下标，时间复杂度O(1);
  • 插入/删除时间复杂度高O(n);

LinkList(单链表/双链表)

  • Head指针、Tail指针；
  • 插入/删除时间复杂度O(1);
  • 查询时间复杂度O(n);

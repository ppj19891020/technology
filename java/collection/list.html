阐述ArrayList、Vector、LinkList的存储性能和特性？

  1. ArrayList和Vector都是使用数组方式存储数据，查询快插入慢；
  2. vector是线程安全的，性能比较差；
  3. Linklist使用双向链表实现存储（将内存中零散的内存单元通过附加的引用关联起来，形成一个可以按照序号索引的数据结构，内存利用率高），插入速度快；
